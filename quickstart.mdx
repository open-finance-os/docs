## Start Building in 3 Steps

Get your Open Finance integration up and running quickly.

<Steps>
  <Step title="Get API Access">
    ### Register for Sandbox Access

    1. Visit the [Open Finance Sandbox](https://sandbox.openfinance.ae)
    2. Register for a developer account
    3. Obtain your API credentials:
       - Client ID
       - Client Secret
       - API Key

    <Note>
      Save your credentials securely - you'll need them for all API calls.
    </Note>
  </Step>

  <Step title="Set Up Your Environment">
    ### Install Development Tools

    ```bash
    # Create a new project
    mkdir my-openfinance-app
    cd my-openfinance-app

    # Initialize your project
    npm init -y

    # Install dependencies
    npm install axios dotenv
    ```

    ### Configure Environment Variables

    Create a `.env` file:
    ```bash
    OPENFINANCE_CLIENT_ID=your_client_id
    OPENFINANCE_CLIENT_SECRET=your_client_secret
    OPENFINANCE_API_KEY=your_api_key
    OPENFINANCE_BASE_URL=https://sandbox.openfinance.ae/api/v1
    ```

    <Tip>
      Never commit your `.env` file to version control!
    </Tip>
  </Step>

  <Step title="Make Your First API Call">
    ### Test Authentication

    Create `index.js`:
    ```javascript
    require('dotenv').config();
    const axios = require('axios');

    async function getAccessToken() {
      try {
        const response = await axios.post(
          `${process.env.OPENFINANCE_BASE_URL}/oauth/token`,
          {
            client_id: process.env.OPENFINANCE_CLIENT_ID,
            client_secret: process.env.OPENFINANCE_CLIENT_SECRET,
            grant_type: 'client_credentials'
          }
        );

        console.log('Token obtained:', response.data.access_token);
        return response.data.access_token;
      } catch (error) {
        console.error('Authentication failed:', error.message);
      }
    }

    // Test the connection
    getAccessToken();
    ```

    Run your code:
    ```bash
    node index.js
    ```

    <Check>
      Success! You should see an access token in the console.
    </Check>
  </Step>
</Steps>

## Sample Projects

Get inspired by these starter examples:

```javascript
// Example: Fetch account balance
async function getAccountBalance(token, accountId) {
  const response = await axios.get(
    `${process.env.OPENFINANCE_BASE_URL}/accounts/${accountId}/balances`,
    {
      headers: {
        'Authorization': `Bearer ${token}`,
        'Accept': 'application/json'
      }
    }
  );

  return response.data;
}

// Example: List recent transactions
async function getTransactions(token, accountId) {
  const response = await axios.get(
    `${process.env.OPENFINANCE_BASE_URL}/accounts/${accountId}/transactions`,
    {
      headers: {
        'Authorization': `Bearer ${token}`,
        'Accept': 'application/json'
      },
      params: {
        limit: 10,
        sort: 'desc'
      }
    }
  );

  return response.data;
}
```

<Info>
  **Pro Tip**: Use the sandbox environment to test with realistic mock data before going to production.
</Info>

## Troubleshooting

Common issues and solutions:

| Issue | Solution |
|-------|----------|
| 401 Unauthorized | Check your credentials and token expiration |
| 429 Too Many Requests | Implement rate limiting and exponential backoff |
| CORS errors | Use a backend proxy or configure allowed origins |
| Connection timeout | Check network and increase timeout settings |

<Warning>
  **Security Reminder**: Always validate input, sanitize data, and never expose credentials in client-side code.
</Warning>
