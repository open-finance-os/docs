# React Native Mobile App Guide

Create a mobile banking application using the [Open Finance Hackathon Starter](https://github.com/open-finance-os/open-finance-hackathon-starter) as your API foundation and React Native for the mobile frontend.

<Warning>
  **Mobile Complexity**: React Native requires additional setup and mobile development experience. Consider web-first approach if your team is new to mobile development.
</Warning>

## Mobile App Development Approach

### üîÑ **Recommended Architecture**
1. **Backend**: Use hackathon starter as API server
2. **Mobile**: React Native app that calls your backend
3. **Authentication**: Handle certificates on backend, not mobile
4. **Deployment**: Backend on Railway/Heroku, mobile via Expo

### üöÄ **Setup Steps**

<Steps>
  <Step title="Setup Backend API">
    ```bash
    # First, get the hackathon starter running as your API
    git clone https://github.com/YOUR_USERNAME/open-finance-hackathon-starter.git
    cd open-finance-hackathon-starter
    npm install
    npm run test:connection

    # Deploy to Railway or Heroku for mobile access
    ```
  </Step>

  <Step title="Create React Native App">
    ```bash
    # Create new React Native app with Expo (easier)
    npx create-expo-app OpenFinanceMobile
    cd OpenFinanceMobile

    # Install dependencies for API calls and navigation
    npx expo install axios react-navigation
    ```
  </Step>

  <Step title="Connect to Backend">
    ```javascript
    // services/api.js
    import axios from 'axios';

    const API_BASE = 'https://your-backend.railway.app'; // Your deployed backend

    export const getAccounts = async () => {
      const response = await axios.get(`${API_BASE}/api/accounts`);
      return response.data;
    };
    ```
  </Step>

  <Step title="Build Mobile UI">
    ```javascript
    // screens/AccountsScreen.js
    import React, { useEffect, useState } from 'react';
    import { View, Text, FlatList } from 'react-native';
    import { getAccounts } from '../services/api';

    export default function AccountsScreen() {
      const [accounts, setAccounts] = useState([]);

      useEffect(() => {
        getAccounts().then(setAccounts);
      }, []);

      return (
        <View>
          <FlatList
            data={accounts}
            renderItem={({ item }) => (
              <Text>{item.account_name}: {item.balance.amount}</Text>
            )}
          />
        </View>
      );
    }
    ```
  </Step>

  <Step title="Run and Test">
    ```bash
    # Start the Expo development server
    npx expo start

    # Test on your device using Expo Go app
    # Or use iOS Simulator / Android Emulator
    ```
  </Step>
</Steps>

## Mobile-Specific Considerations

### üîê **Security for Mobile**
- **Never store certificates** on mobile devices
- **Use backend proxy** for all Open Finance API calls
- **Implement biometric authentication** for user security
- **Use secure storage** for session tokens

### üì± **UX Best Practices**
- **Offline support** for account balances
- **Loading states** for API calls
- **Error handling** with user-friendly messages
- **Touch-friendly** button sizes and spacing

### üöÄ **Deployment**
```bash
# Build for app stores
npx expo build:ios
npx expo build:android

# Or publish to Expo for easy testing
npx expo publish
```

### üí∏ **Payment Form**
```javascript
// components/PaymentForm.js
import React, { useState } from 'react';
import { View, TextInput, TouchableOpacity, Text, Alert } from 'react-native';
import { initiatePayment } from '../services/api';

export function PaymentForm({ onSuccess }) {
  const [amount, setAmount] = useState('');
  const [recipient, setRecipient] = useState('');
  const [loading, setLoading] = useState(false);

  const handleSubmit = async () => {
    if (!amount || !recipient) {
      Alert.alert('Error', 'Please fill all fields');
      return;
    }

    setLoading(true);
    try {
      const result = await initiatePayment({
        amount: parseFloat(amount),
        recipient,
        reference: `Mobile payment ${Date.now()}`,
      });
      Alert.alert('Success', 'Payment initiated successfully');
      onSuccess(result);
    } catch (error) {
      Alert.alert('Error', 'Payment failed. Please try again.');
    } finally {
      setLoading(false);
    }
  };

  return (
    <View style={styles.form}>
      <TextInput
        style={styles.input}
        placeholder="Amount"
        value={amount}
        onChangeText={setAmount}
        keyboardType="decimal-pad"
      />
      <TextInput
        style={styles.input}
        placeholder="Recipient Account"
        value={recipient}
        onChangeText={setRecipient}
      />
      <TouchableOpacity
        style={[styles.button, loading && styles.buttonDisabled]}
        onPress={handleSubmit}
        disabled={loading}
      >
        <Text style={styles.buttonText}>
          {loading ? 'Processing...' : 'Send Payment'}
        </Text>
      </TouchableOpacity>
    </View>
  );
}
```
