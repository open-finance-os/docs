## Quick Setup

<Steps>
  <Step title="Fork & Clone">
    ```bash
    git clone https://github.com/YOUR_USERNAME/template-nodejs-express.git
    cd template-nodejs-express
    ```
  </Step>

  <Step title="Install Dependencies">
    ```bash
    npm install
    ```
  </Step>

  <Step title="Environment Setup">
    ```bash
    cp .env.example .env

    # Edit .env with your credentials
    OPENFINANCE_CLIENT_ID=your_client_id
    OPENFINANCE_CLIENT_SECRET=your_client_secret
    DATABASE_URL=postgresql://user:password@localhost:5432/openfinance
    REDIS_URL=redis://localhost:6379
    JWT_SECRET=your_jwt_secret
    ```
  </Step>

  <Step title="Database Setup">
    ```bash
    # Start PostgreSQL and Redis (using Docker)
    docker-compose up -d db redis

    # Run database migrations
    npx prisma migrate dev

    # Seed with sample data
    npm run db:seed
    ```
  </Step>

  <Step title="Start Development Server">
    ```bash
    npm run dev
    ```

    API will be available at `http://localhost:8000`
  </Step>
</Steps>

## API Endpoints

### Authentication
```typescript
POST /auth/login          # User login
POST /auth/refresh        # Refresh JWT token
POST /auth/logout         # User logout
```

### Accounts
```typescript
GET    /accounts          # List user accounts
GET    /accounts/:id      # Get account details
GET    /accounts/:id/balance  # Get account balance
GET    /accounts/:id/transactions  # Get transactions
```

### Payments
```typescript
POST   /payments          # Initiate payment
GET    /payments/:id      # Get payment status
PUT    /payments/:id/confirm  # Confirm payment
DELETE /payments/:id      # Cancel payment
```

### Example API Usage
```typescript
// Get account balance
const response = await fetch('/accounts/123/balance', {
  headers: {
    'Authorization': `Bearer ${token}`,
    'Content-Type': 'application/json'
  }
});

const balance = await response.json();
```

## Deployment

### Railway (Recommended)
```bash
# Install Railway CLI
npm install -g @railway/cli

# Login and deploy
railway login
railway init
railway up
```

### Heroku
```bash
# Install Heroku CLI and deploy
heroku create your-app-name
git push heroku main
```

### Docker
```bash
# Build and run container
docker build -t openfinance-api .
docker run -p 8000:8000 openfinance-api
```
