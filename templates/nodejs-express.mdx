## Express Conversion Steps

<Steps>
  <Step title="Start with Hackathon Starter">
    ```bash
    # Clone the starter repository
    git clone https://github.com/YOUR_USERNAME/open-finance-hackathon-starter.git
    cd open-finance-hackathon-starter

    # Test the existing examples
    npm install
    npm run test:connection
    npm run example:auth
    ```
  </Step>

  <Step title="Add Express Framework">
    ```bash
    # Install Express and middleware
    npm install express helmet cors express-rate-limit
    npm install -D @types/express @types/cors
    ```
  </Step>

  <Step title="Create Express Server">
    ```javascript
    // server.js
    const express = require('express');
    const helmet = require('helmet');
    const cors = require('cors');
    const rateLimit = require('express-rate-limit');
    const { getAccessToken } = require('./examples/auth');

    const app = express();

    app.use(helmet());
    app.use(cors());
    app.use(express.json());
    app.use(rateLimit({ windowMs: 15 * 60 * 1000, max: 100 }));

    // Convert examples to API routes
    app.get('/api/accounts', async (req, res) => {
      // Use code from examples/accounts.js
    });

    app.listen(3000, () => console.log('Server running on port 3000'));
    ```
  </Step>

  <Step title="Convert Examples to Routes">
    Transform the existing examples into Express routes using the working code as a foundation.
  </Step>
</Steps>

## API Endpoints

### Authentication
```typescript
POST /auth/login          # User login
POST /auth/refresh        # Refresh JWT token
POST /auth/logout         # User logout
```

### Accounts
```typescript
GET    /accounts          # List user accounts
GET    /accounts/:id      # Get account details
GET    /accounts/:id/balance  # Get account balance
GET    /accounts/:id/transactions  # Get transactions
```

### Payments
```typescript
POST   /payments          # Initiate payment
GET    /payments/:id      # Get payment status
PUT    /payments/:id/confirm  # Confirm payment
DELETE /payments/:id      # Cancel payment
```

### Example API Usage
```typescript
// Get account balance
const response = await fetch('/accounts/123/balance', {
  headers: {
    'Authorization': `Bearer ${token}`,
    'Content-Type': 'application/json'
  }
});

const balance = await response.json();
```

## Deployment

### Railway (Recommended)
```bash
# Install Railway CLI
npm install -g @railway/cli

# Login and deploy
railway login
railway init
railway up
```

### Heroku
```bash
# Install Heroku CLI and deploy
heroku create your-app-name
git push heroku main
```

### Docker
```bash
# Build and run container
docker build -t openfinance-api .
docker run -p 8000:8000 openfinance-api
```
