## Quick Start Guide

<Steps>
  <Step title="Fork the Starter Repository">
    Visit the [Open Finance Hackathon Starter](https://github.com/open-finance-os/open-finance-hackathon-starter) and fork it to your GitHub account.
  </Step>

  <Step title="Clone & Setup">
    ```bash
    # Clone your forked repository
    git clone https://github.com/YOUR_USERNAME/open-finance-hackathon-starter.git
    cd open-finance-hackathon-starter

    # Install dependencies
    npm install
    ```
  </Step>

  <Step title="Environment Configuration">
    ```bash
    # Copy environment template
    cp .env.example .env

    # Add your credentials (provided by Nebras)
    OPENFINANCE_CLIENT_ID=your_client_id
    OPENFINANCE_CLIENT_SECRET=your_client_secret
    OPENFINANCE_BASE_URL=https://sandbox.openfinance.ae
    TRANSPORT_CERT_PATH=./certs/transport.pem
    TRANSPORT_KEY_PATH=./certs/transport-key.pem
    ```
  </Step>

  <Step title="Test Connection">
    ```bash
    # Test your API connection
    npm run test:connection

    # Run authentication example
    npm run example:auth
    ```
  </Step>

  <Step title="Choose Your Framework">
    Adapt the starter for your preferred technology stack using our framework guides, or continue with the Node.js examples provided.
  </Step>
</Steps>
