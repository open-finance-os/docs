## Quick Start Guide

<Steps>
  <Step title="Choose Your Template">
    Pick the technology stack you're most comfortable with. All templates provide the same Open Finance functionality.
  </Step>

  <Step title="Fork & Clone">
    ```bash
    # Fork the repository on GitHub, then:
    git clone https://github.com/YOUR_USERNAME/TEMPLATE_NAME.git
    cd TEMPLATE_NAME
    ```
  </Step>

  <Step title="Environment Setup">
    ```bash
    # Copy environment template
    cp .env.example .env

    # Add your API credentials
    OPENFINANCE_CLIENT_ID=your_client_id
    OPENFINANCE_CLIENT_SECRET=your_client_secret
    OPENFINANCE_API_KEY=your_api_key
    ```
  </Step>

  <Step title="Install & Run">
    ```bash
    # Install dependencies
    npm install  # or pip install -r requirements.txt

    # Start development server
    npm run dev  # or python main.py
    ```
  </Step>

  <Step title="Verify Integration">
    Open your browser and test the API connection. You should see live data from the Open Finance sandbox.
  </Step>
</Steps>
