## Code Examples

### ğŸ¦ **Converting Account Management**
Adapt from `examples/accounts.js` in the starter:

```typescript
// lib/accounts.ts (converted from starter)
import { getAccessToken } from './auth';

export async function getAccounts() {
  const { token } = await getAccessToken();

  const response = await fetch(`${process.env.OPENFINANCE_BASE_URL}/accounts`, {
    headers: {
      'Authorization': `Bearer ${token}`,
      'Accept': 'application/json'
    }
  });

  return response.json();
}

// components/AccountCard.tsx
export function AccountCard({ account }: { account: Account }) {
  return (
    <div className="p-4 border rounded-lg">
      <h3>{account.account_name}</h3>
      <p className="text-lg font-bold">{account.balance.amount} {account.balance.currency}</p>
    </div>
  );
}
```

### ğŸ’¸ **Payment Flow Component**
Based on `examples/payments.js`:

```typescript
// components/PaymentForm.tsx
import { useForm } from 'react-hook-form';
import { initiatePayment } from '../lib/payments';

export function PaymentForm() {
  const { register, handleSubmit } = useForm();

  const onSubmit = async (data: any) => {
    try {
      const result = await initiatePayment(data);
      console.log('Payment initiated:', result);
    } catch (error) {
      console.error('Payment failed:', error);
    }
  };

  return (
    <form onSubmit={handleSubmit(onSubmit)} className="space-y-4">
      <input {...register('amount')} placeholder="Amount" />
      <input {...register('recipient')} placeholder="Recipient" />
      <button type="submit">Send Payment</button>
    </form>
  );
}
```

### ğŸ” **Authentication Hook**
Convert the starter's auth example:

```typescript
// hooks/useAuth.ts
import { useState, useEffect } from 'react';
import { getAccessToken } from '../lib/auth';

export function useAuth() {
  const [token, setToken] = useState<string | null>(null);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    async function authenticate() {
      try {
        const { token } = await getAccessToken();
        setToken(token);
      } catch (error) {
        console.error('Authentication failed:', error);
      } finally {
        setLoading(false);
      }
    }

    authenticate();
  }, []);

  return { token, loading };
}
```

## Key Adaptations

### ğŸ¨ **Client-Side API Calls**
Unlike the starter's Node.js server-side examples, React components need client-side API calls:

```typescript
// Convert server-side axios to client-side fetch
// Replace httpsAgent (Node.js specific) with simple fetch
const response = await fetch(url, {
  headers: {
    'Authorization': `Bearer ${token}`,
    'Content-Type': 'application/json'
  }
});
```

### ğŸ”§ **Environment Variables**
Convert Node.js environment variables to Next.js format:

```bash
# .env.local (Next.js format)
NEXT_PUBLIC_OPENFINANCE_BASE_URL=https://sandbox.openfinance.ae
OPENFINANCE_CLIENT_ID=your_client_id
OPENFINANCE_CLIENT_SECRET=your_client_secret
```

### ğŸ“± **Certificate Handling**
Certificates are server-side only in Node.js. For client-side React:
- API calls go through Next.js API routes
- Or use a backend proxy for certificate-based authentication
