## Quick Setup

<Steps>
  <Step title="Fork & Clone">
    ```bash
    # Fork the repository, then clone it
    git clone https://github.com/YOUR_USERNAME/template-react-nextjs.git
    cd template-react-nextjs
    ```
  </Step>

  <Step title="Install Dependencies">
    ```bash
    npm install
    # or
    yarn install
    ```
  </Step>

  <Step title="Environment Setup">
    ```bash
    # Copy environment template
    cp .env.example .env.local

    # Add your credentials to .env.local
    OPENFINANCE_CLIENT_ID=your_client_id
    OPENFINANCE_CLIENT_SECRET=your_client_secret
    OPENFINANCE_API_KEY=your_api_key
    OPENFINANCE_BASE_URL=https://sandbox.openfinance.ae/api/v1
    NEXTAUTH_SECRET=your_nextauth_secret
    NEXTAUTH_URL=http://localhost:3000
    ```
  </Step>

  <Step title="Start Development Server">
    ```bash
    npm run dev
    # or
    yarn dev
    ```

    Open [http://localhost:3000](http://localhost:3000) to see your app.
  </Step>

  <Step title="Test API Connection">
    Navigate to the dashboard and verify that account data loads from the Open Finance sandbox.
  </Step>
</Steps>

### Adding Authentication Guards
```typescript
// Protect routes with authentication
export default function ProtectedLayout({
  children,
}: {
  children: React.ReactNode;
}) {
  const { data: session, status } = useSession();

  if (status === "loading") return <Loading />;
  if (!session) redirect("/login");

  return <>{children}</>;
}
```

## Troubleshooting

### Common Issues

<AccordionGroup>
  <Accordion title="API Connection Fails" icon="wifi">
    **Solutions:**
    - Check environment variables are set correctly
    - Verify API credentials in sandbox
    - Check network connectivity
    - Review browser console for CORS issues
  </Accordion>

  <Accordion title="Build Errors" icon="exclamation-triangle">
    **Solutions:**
    - Clear `.next` folder: `rm -rf .next`
    - Clear node modules: `rm -rf node_modules && npm install`
    - Check TypeScript errors: `npm run type-check`
    - Verify all environment variables are set
  </Accordion>

  <Accordion title="Styling Issues" icon="paintbrush">
    **Solutions:**
    - Ensure Tailwind is configured correctly
    - Check if custom CSS conflicts with Tailwind
    - Verify component imports from shadcn/ui
    - Test in different browsers
  </Accordion>
</AccordionGroup>
