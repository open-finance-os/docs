Accelerate your development with Model Context Protocol (MCP) servers that integrate directly with Claude Code for enhanced productivity.

## What are MCP Servers?

MCP servers extend Claude Code's capabilities by providing direct access to external tools and services. Think of them as **AI-powered integrations** that let Claude Code interact with your development environment, databases, cloud services, and more.

### **Hackathon Benefits**
- **50-70% faster development** with automated workflows
- **Professional-grade infrastructure** from day one
- **Seamless tool integration** without manual context switching
- **AI-enhanced debugging** and optimization
- **Real-time collaboration** across team members

## Essential MCP Servers for Hackathon Teams

### üöÄ **Must-Have (Essential)**

<AccordionGroup>
  <Accordion title="GitHub MCP Server" icon="github">
    **Use Case**: Repository management, issue tracking, code navigation

    **Setup (Tested & Working):**
    ```bash
    # Install GitHub MCP Server
    claude mcp add github npx @modelcontextprotocol/server-github

    # Verify installation
    claude mcp list
    claude mcp get github
    ```

    **What You'll See:**
    ```
    ‚úì Connected github: npx @modelcontextprotocol/server-github
    ```

    **Real Hackathon Usage:**
    - Browse repository files and structure
    - Search through code and documentation
    - Navigate project history and changes
    - Understand codebase context for AI assistance

    **Team Benefits:**
    - Enhanced code understanding for AI assistance
    - Better context for code generation
    - Improved navigation of project structure
    - Faster onboarding for team members

    <Warning>
      **Note**: Some advanced GitHub operations may require additional configuration. For basic repository browsing and code context, the default setup works immediately.
    </Warning>
  </Accordion>

## Troubleshooting Common Issues

### **MCP Connection Problems**
```bash
# Check MCP server status
claude mcp list

# Get detailed info about a specific server
claude mcp get github

# Remove and re-add problematic server
claude mcp remove github
claude mcp add github npx @modelcontextprotocol/server-github
```

### **Deprecation Warnings**
**Issue**: You may see warnings like "Package no longer supported"

**Solution**: These warnings are normal and don't affect functionality. The MCP servers continue to work despite deprecation warnings.

```bash
# Example warning you might see:
npm warn deprecated @modelcontextprotocol/server-github@2025.4.8: Package no longer supported

# This is expected - the server still works
```

### **Path Issues (Filesystem MCP)**
**Issue**: Filesystem MCP can't access your project files

**Solution**: Ensure the path is correct and accessible
```bash
# Check your current directory
pwd

# Use absolute path when adding filesystem MCP
claude mcp add filesystem npx @modelcontextprotocol/server-filesystem /full/path/to/project

# For hackathon teams, typically:
claude mcp add filesystem npx @modelcontextprotocol/server-filesystem /Users/yourname/hackathon-project
```
