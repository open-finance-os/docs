- Create consents with the AlTareq Model Bank  
- Obtain access tokens

## Setting up Postman

### 1. Download the Postman Collection  

<a href="/hackathon-guide/json/adcb-nebras-hackathon-postman.json" download class="button">Download Collection</a>

### 2. Import into Postman  

Open **Postman**, click **Import**, and select the downloaded file `adcb-nebras-hackathon-postman.json`.


### 3. Add Client Credentials  

Add the **client transport certificate** and **client transport private key** from your **Client Pack** to the Postman requests. This ensures secure communication with the APIs.
