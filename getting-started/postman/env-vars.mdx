As part of your Client Pack, you will have received a file named **postman_environment.json**.  
This file contains all the required environment variables, preconfigured for your use.

### Step 1 - import postman_environment.json
In Postman, open the **Environments** section and import the file **postman_environment.json**.

### Step 2 - set Hackathon Environment
Set the active environment to **Hackathon Environment**.
