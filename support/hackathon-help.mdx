### Code Examples

#### Authentication Flow

```javascript
import { OpenFinanceClient } from '@openfinance/sdk';

const client = new OpenFinanceClient({
  apiKey: process.env.API_KEY,
  sandbox: true
});

async function authenticate() {
  try {
    const token = await client.auth.getAccessToken({
      scope: ['accounts', 'payments'],
      redirectUri: 'http://localhost:3000/callback'
    });

    return token;
  } catch (error) {
    console.error('Authentication failed:', error);
  }
}
```

#### Fetching Account Data

```javascript
async function getAccountInfo(accountId) {
  const response = await client.accounts.get(accountId);

  return {
    balance: response.balance,
    transactions: response.transactions.slice(0, 10),
    accountHolder: response.accountHolder
  };
}
```

## Common Challenges & Solutions

### Troubleshooting Guide

<Warning>
  **Common Issues and Quick Fixes**
</Warning>

| Issue | Solution |
|-------|----------|
| API Key not working | Check sandbox environment and key permissions |
| CORS errors | Use proxy in development or configure allowed origins |
| Rate limiting | Implement exponential backoff and caching |
| Data format issues | Review API documentation for expected formats |
| Authentication failures | Verify token expiration and refresh logic |
